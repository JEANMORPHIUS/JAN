# Contact Setup Guide

**Quick guide for setting up contact information.**

---

## Email Options

### Recommended: `info@siyem.org`

**Pros:**
- Most common pattern (info@domain.com)
- Professional and clear
- Already referenced in codebase
- Easy to remember

**Cons:**
- Very generic (might get spam)

### Alternative: `hello@siyem.org`

**Pros:**
- Friendly, welcoming tone
- Modern, approachable
- Good for community projects

**Cons:**
- Less formal
- Might seem too casual for some

### Alternative: `contact@siyem.org`

**Pros:**
- Professional and formal
- Clear purpose
- Standard business pattern

**Cons:**
- Very generic
- Less personal

## Discord Setup Steps

### 1. Create Discord Server

1. Open Discord
2. Click "+" (Add Server)
3. Choose "Create My Own"
4. Name it: "JAN Studio Community" or "The Table"
5. Choose region (closest to you)

### 2. Set Up Channels

**Essential Channels:**
- `#general` - General community chat
- `#oracle` - Oracle SIYEM discussions
- `#personas` - Persona sharing and examples
- `#help` - Technical support
- `#announcements` - Project updates

**Optional Channels:**
- `#book-of-racon` - 40 Laws discussions
- `#campaign-automation` - Campaign tools
- `#showcase` - Share what you've created
- `#off-topic` - Casual chat

### 3. Get Invite Link

1. Right-click server name ‚Üí "Invite People"
2. Set expiration: "Never expire" (or 7 days for security)
3. Set max uses: Unlimited (or limit for security)
4. Copy invite link
5. Update README.md with link

### 4. Set Up Roles (Optional)

- `@Contributors` - Active contributors
- `@Maintainers` - Project maintainers
- `@Community` - General members

## Email Forwarding Setup

### If Using Google Workspace (siyem.org)

1. Go to Google Admin Console
2. Users ‚Üí Add user: `info@siyem.org`
3. Or set up email forwarding to your main email
4. Configure email routing

### If Using Other Email Service

- **Gmail**: Set up email alias
- **ProtonMail**: Use custom domain
- **FastMail**: Set up forwarding
- **Simple**: Use email forwarding service
