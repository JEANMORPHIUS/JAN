## FILE NAMING CONVENTION

### Video Files
```
EDIBLE_LONDON_[SEGMENT]_[SHOT]_[DATE].mp4
Example: EDIBLE_LONDON_SEG1_SHOT1_20240115.mp4
```

### Audio Files
```
EDIBLE_LONDON_[TYPE]_[DESCRIPTION]_[DATE].wav
Example: EDIBLE_LONDON_VO_MainNarration_20240115.wav
```

### Graphics Files
```
EDIBLE_LONDON_[TYPE]_[DESCRIPTION]_[DATE].[ext]
Example: EDIBLE_LONDON_ICON_Gemini_20240115.png
```

## ASSET REQUIREMENTS

### Video Specifications
- **Format:** MP4 (H.264 or H.265)
- **Resolution:** 1080x1920 (vertical)
- **Frame Rate:** 30fps
- **Color Space:** Rec.709 (for delivery)

### Audio Specifications
- **Format:** WAV (for editing), AAC (for delivery)
- **Sample Rate:** 48kHz
- **Bit Depth:** 24-bit (for editing), 16-bit (for delivery)
- **Channels:** Stereo

### Graphics Specifications
- **Format:** PNG (with transparency for overlays), MOV/MP4 (for animations)
- **Resolution:** Match video resolution (1080x1920)
- **Color Profile:** sRGB (for stills), Rec.709 (for animations)
