### Entity Content API

- `/entity-content/status` - System status
- `/entity-content/entities` - All entities (language parameter)
- `/entity-content/entities/{entity_name}` - Entity content (language parameter)
- `/entity-content/entities/{entity_name}/bilingual` - Entity content (both languages)
- `/entity-content/report` - Complete entity content report

### UI Refinement API

- `/ui-refinement/components/{key}` - Component text (language parameter)

- `/ui-refinement/report` - Complete UI refinement report
