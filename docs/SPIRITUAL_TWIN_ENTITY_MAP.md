## 10. HOW TO USE THIS MAP

- **Add a new entity (any scope):** Assign a spiritual twin (or pillar); add a row here and an entry in [data/spiritual_twin_entities.json](../data/spiritual_twin_entities.json).  
- **Family name:** Add to [SPIRITUAL_DNA_FAMILY_TREE.md](../SPIRITUAL_DNA_FAMILY_TREE.md) and [FAMILY_TREE_LINEAGE_DATA.json](../FAMILY_TREE_LINEAGE_DATA.json); twin = lineage branch or spiritual contract.
- **Partner / external:** Add to [data/business_entities_leadership.json](../data/business_entities_leadership.json); twin = venture, family entity, or pillar.
- **Link from a profile:** In the entity’s `profile.md` or entity JSON, add `spiritual_twin: <twin_id>` and link to this map.  
- **Cursor / AI:** [docs/FAMILY_INGESTION_SYNC.md](FAMILY_INGESTION_SYNC.md) and [DOCUMENTATION_INDEX.md](../DOCUMENTATION_INDEX.md) reference this map so the whole graph is visible.
