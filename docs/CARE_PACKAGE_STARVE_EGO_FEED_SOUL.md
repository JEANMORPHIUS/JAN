### 2. Health Tracking - Track Any Condition

**What it does:** Works for ANY condition, illness, or disease.

**How to use:**

**Python:**
```python
from scripts.unified_global_access import UnifiedGlobalAccess

access = UnifiedGlobalAccess(user_id="my_user")

# Register condition
access.register_my_condition(
    condition_name="Type 1 Diabetes",
    category="metabolic",
    primary_metrics=["blood_glucose", "insulin_units"]
)

# Log health entry (morning loop example)
access.log_my_health(
    metrics=[
        {
            "metric_type": "lab_result",
            "metric_name": "blood_glucose",
            "value": 20.8,
            "unit": "mmol/L"
        },
        {
            "metric_type": "medication",
            "metric_name": "Degludec",
            "value": 11,
            "unit": "units"
        }
    ],
    entry_type="routine",
    condition_name="Type 1 Diabetes",
    notes="Morning loop: 500ml mixed with 500ml Ribena, Shilajit salt, insulin"
)
```

**API:**
```bash
# Register condition
curl -X POST http://localhost:8000/api/health/condition \
  -H "Content-Type: application/json" \
  -d '{
    "condition_name": "Type 1 Diabetes",
    "category": "metabolic",
    "primary_metrics": ["blood_glucose", "insulin_units"]
  }'

# Log entry
curl -X POST http://localhost:8000/api/health/log \
  -H "Content-Type: application/json" \
  -d '{
    "metrics": [
      {
        "metric_type": "lab_result",
        "metric_name": "blood_glucose",
        "value": 20.8,
        "unit": "mmol/L"
      }
    ],
    "entry_type": "routine",
    "condition_name": "Type 1 Diabetes"
  }'
```

**What you get:**
- Condition tracking
- Health metric logging
- Pattern detection
- Data export
- Summary reports
