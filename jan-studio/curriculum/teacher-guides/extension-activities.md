# Extension Activities

**Additional activities to extend learning beyond the core curriculum**

---

## Activity 1: Persona Gallery

**Objective:** Create a class collection of personas

**Duration:** 30-45 minutes

**Materials:**
- JAN Studio
- Display board or digital gallery
- Persona cards/templates

**Instructions:**
1. Each student creates their best persona
2. Students create a "persona card" with:
   - Name and description
   - Example output
   - Why they created it
3. Display all personas in a gallery
4. Students can try each other's personas
5. Vote on favorites (optional)

**Learning Outcomes:**
- Appreciation for different approaches
- Understanding of persona diversity
- Collaboration and sharing

---

## Activity 2: Persona Swap Challenge

**Objective:** Use someone else's persona and improve it

**Duration:** 30 minutes

**Materials:**
- JAN Studio
- Persona swap worksheet

**Instructions:**
1. Students pair up
2. Each student uses their partner's persona
3. Generate content with it
4. Provide feedback and suggestions
5. Partner improves persona based on feedback
6. Compare original vs. improved

**Learning Outcomes:**
- Perspective-taking
- Giving and receiving feedback
- Iterative improvement

---

## Activity 3: Content Collection Book

**Objective:** Create a class book of generated content

**Duration:** Multiple sessions

**Materials:**
- JAN Studio
- Print capability
- Binding materials (optional)

**Instructions:**
1. Each student contributes their best work
2. Organize by theme or persona type
3. Create table of contents
4. Print and bind (or create digital book)
5. Share with class, school, or parents

**Learning Outcomes:**
- Purpose in collective work
- Understanding of publication process
- Sharing with wider audience

---

## Activity 4: Persona for a Purpose

**Objective:** Create personas for specific real-world needs

**Duration:** 45 minutes

**Materials:**
- JAN Studio
- Real-world scenario cards

**Scenarios:**
- Create a persona to help explain science to younger students
- Create a persona to write stories for a children's hospital
- Create a persona to help with homework explanations
- Create a persona to write encouraging messages

**Instructions:**
1. Students choose or are assigned a scenario
2. Create persona specifically for that purpose
3. Test persona with relevant prompts
4. Present persona and explain how it meets the need

**Learning Outcomes:**
- Real-world application
- Understanding audience needs
- Purpose-driven design

---

## Activity 5: Rule Experimentation

**Objective:** Understand how rules affect output

**Duration:** 30 minutes

**Materials:**
- JAN Studio
- Experiment worksheet

**Instructions:**
1. Start with a basic persona
2. Generate content (baseline)
3. Add one rule, generate again
4. Compare results
5. Repeat with different rules
6. Document findings

**Learning Outcomes:**
- Scientific thinking
- Understanding cause and effect
- Documentation skills

---

## Activity 6: Prompt Library

**Objective:** Create a class library of effective prompts

**Duration:** Ongoing

**Materials:**
- JAN Studio
- Shared document or board

**Instructions:**
1. Students contribute their best prompts
2. Categorize by type (story, explanation, creative)
3. Test prompts and rate effectiveness
4. Create a searchable library
5. Students can use library for inspiration

**Learning Outcomes:**
- Resource creation
- Collaboration
- Understanding of effective prompts

---

## Activity 7: Persona Evolution Timeline

**Objective:** Track how personas improve over time

**Duration:** Throughout curriculum

**Materials:**
- JAN Studio
- Timeline template

**Instructions:**
1. Document persona at each lesson
2. Note changes and improvements
3. Compare outputs at each stage
4. Create visual timeline
5. Reflect on growth

**Learning Outcomes:**
- Self-reflection
- Understanding of improvement process
- Documentation skills

## Activity 8: Cross-Curricular Integration

**Objective:** Use personas in other subjects

**Duration:** Varies by subject

**Examples:**

**Science:**
- Create persona to explain scientific concepts
- Generate explanations for experiments
- Create science stories

**History:**
- Create historical storyteller persona
- Generate stories from different time periods
- Explain historical events

**Language Arts:**
- Create different writing style personas
- Generate examples of different genres
- Practice writing techniques

**Math:**
- Create persona to explain math concepts
- Generate word problems
- Explain problem-solving steps

**Learning Outcomes:**
- Application across subjects
- Deeper understanding of concepts
- Creative expression

## Activity 9: Persona Showcase Event

**Objective:** Share work with wider community

**Duration:** 1-2 hours

**Materials:**
- JAN Studio
- Display materials
- Refreshments (optional)

**Instructions:**
1. Students prepare presentations
2. Set up showcase stations
3. Invite parents, other classes, or community
4. Students demonstrate their personas
5. Share generated content
6. Answer questions

**Learning Outcomes:**
- Public speaking
- Purpose in work
- Community engagement

---

## Activity 10: Advanced Persona Features

**Objective:** Explore advanced persona capabilities

**Duration:** 45 minutes

**Materials:**
- JAN Studio
- Advanced features guide

**Features to Explore:**
- Multiple personas working together
- Persona templates
- Custom validation rules
- Integration with other tools

**Instructions:**
1. Introduce advanced features
2. Students experiment with features
3. Document findings
4. Share discoveries

**Learning Outcomes:**
- Advanced skills
- Exploration and discovery
- Technical understanding

---

## Implementation Tips

1. **Choose activities** that fit your class needs
2. **Adapt activities** for your students' level
3. **Allow choice** - let students pick activities
4. **Celebrate work** - showcase student creations
5. **Connect to curriculum** - link to learning objectives

---

## Assessment

Extension activities can be:
- **Formative:** Check understanding
- **Summative:** Demonstrate mastery
- **Self-directed:** Student choice
- **Collaborative:** Group work
