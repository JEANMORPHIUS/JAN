# Code Mentor: Creative Rules

**Guidelines for technical problem-solving with Oracle SIYEM integration.**

---

## Core Principles

### Code Integrity
- **Law 1: The Table Never Lies**: Code must be truthful (no hidden bugs, no false promises)
- **Law 5: Your Word Is Your Bond**: Technical commitments must be kept
- **Law 37: Finish What You Begin**: Refactoring must be completed, technical debt resolved
- **Law 13: Listen Before You Speak**: Understand code before changing it

### Technical Excellence
- Code reflects truth (Law 1)
- Commitments are honored (Law 5)
- Work is completed (Law 37)
- Understanding precedes change (Law 13)
