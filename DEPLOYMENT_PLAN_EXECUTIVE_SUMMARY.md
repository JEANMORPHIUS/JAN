## TECHNICAL REQUIREMENTS

### Platform Components
1. **Learning Management System (LMS)**
   - Student portal (lesson viewing, quizzes, progress tracking)
   - Teacher dashboard (assign lessons, grade assessments, analytics)
   - Admin panel (school management, user management)
   - Parent portal (progress viewing, communication)

2. **Content Delivery**
   - 376 lessons (text, images, audio)
   - Bilingual UI (Turkish/English)
   - Mobile-responsive design
   - Fast media delivery (CDN)

3. **Payment & Subscriptions**
   - Stripe integration (Turkish Lira, Euro, USD)
   - School, teacher, student licenses
   - Invoicing and billing
   - Subscription management

4. **Analytics & Reporting**
   - Student progress tracking
   - Teacher effectiveness metrics
   - School-wide dashboards
   - Engagement analytics

### Technology Stack
- **Frontend:** React + TypeScript + Tailwind CSS + Next.js
- **Backend:** Node.js + Express (API), Python (SIYEM services)
- **Database:** PostgreSQL (relational), MongoDB (content), Redis (caching)
- **Infrastructure:** AWS or Digital Ocean, Docker, Kubernetes, Cloudflare CDN
- **Security:** TLS 1.3, JWT auth, bcrypt, data encryption, GDPR/KVKK compliance
