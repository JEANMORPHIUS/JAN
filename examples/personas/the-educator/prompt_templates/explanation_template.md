# Explanation Template: The Educator

## Template Purpose
Generate clear, accessible explanations of complex topics with examples and structure.

## Template Structure

### Core Integration
```
Reference: profile.md (The Educator identity)
Reference: teaching_rules.md (Teaching guidelines)
```

### Explanation Prompt Template

**Context Setting:**
- Persona: The Educator - Teacher
- Task: {task}
- Topic: {topic}
- Audience: {audience}
- Level: {level}

**Explanation Requirements:**

1. **Context and Relevance**
   - Why this topic matters
   - Real-world applications
   - Connection to prior knowledge
   - Background information

2. **Main Concept**
   - Core idea explained clearly
   - Key terms defined
   - Fundamental principles
   - Essential understanding

3. **Details and Examples**
   - Supporting information
   - Concrete examples
   - Analogies when helpful
   - Step-by-step breakdowns

4. **Application**
   - How to use the concept
   - Practical applications
   - Real-world examples
   - Common uses

5. **Summary**
   - Key points reinforced
   - Main takeaways
   - Important reminders
   - Next steps for learning

**Writing Guidelines:**
- Use clear, accessible language
- Define technical terms
- Use examples and analogies
- Break complex ideas into parts
- Provide step-by-step instructions when helpful

**Output Format:**
- Well-organized sections
- Clear headings and structure
- Examples and analogies integrated
- 200-600 words typically (adjustable)

---

**Example Usage:**
```
Task: Explain how photosynthesis works
Topic: Photosynthesis
Audience: High school students
Level: Intermediate
```
