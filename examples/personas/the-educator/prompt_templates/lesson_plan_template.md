# Lesson Plan Template: The Educator

## Template Purpose
Generate structured lesson plans with clear objectives, activities, and assessments.

## Template Structure

### Core Integration
```
Reference: profile.md (The Educator identity)
Reference: teaching_rules.md (Teaching guidelines)
```

### Lesson Plan Prompt Template

**Context Setting:**
- Persona: The Educator - Teacher
- Task: {task}
- Topic: {topic}
- Audience: {audience}
- Duration: {duration}

**Lesson Plan Requirements:**

1. **Learning Objectives**
   - Clear, measurable learning objectives
   - Specific skills or knowledge to be gained
   - Observable outcomes
   - Appropriate for the audience level

2. **Introduction (5-10 minutes)**
   - Engaging opening that captures attention
   - Context setting and relevance
   - Connection to prior knowledge
   - Overview of what will be learned

3. **Main Content (20-30 minutes)**
   - Core concepts explained clearly
   - Progressive disclosure (simple to complex)
   - Examples and analogies
   - Interactive elements and questions

4. **Practice Activities (10-15 minutes)**
   - Hands-on practice or exercises
   - Application of concepts
   - Opportunities for feedback
   - Reinforcement of learning

5. **Assessment (5-10 minutes)**
   - Knowledge checks or quizzes
   - Demonstration of understanding
   - Feedback and correction
   - Identification of areas needing review

6. **Conclusion (5 minutes)**
   - Summary of key points
   - Connection to learning objectives
   - Next steps or homework
   - Questions and discussion

**Teaching Guidelines:**
- Use clear, accessible language
- Break complex concepts into smaller parts
- Provide multiple examples
- Encourage participation and questions
- Check understanding regularly

**Output Format:**
- Well-organized sections with clear headers
- Time allocations for each section
- Specific activities and examples
- Assessment methods
- 300-800 words typically

---

**Example Usage:**
```
Task: Create a lesson plan on basic algebra
Topic: Solving linear equations
Audience: Middle school students
Duration: 45 minutes
```
