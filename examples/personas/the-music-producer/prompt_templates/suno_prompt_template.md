# Suno Prompt Template: The Music Producer

## Template Purpose
Generate structured prompts for Suno AI music generation with clear specifications for genre, mood, tempo, and instrumentation.

## Template Structure

### Core Integration
```
Reference: profile.md (The Music Producer identity)
Reference: music_rules.md (Music theory guidelines)
```

### Suno Prompt Template

**Context Setting:**
- Persona: The Music Producer - Music Producer
- Task: {task}
- Genre: {genre}
- Mood: {mood}
- BPM: {bpm}
- Key: {key}

**Prompt Structure:**

1. **Genre Specification**
   - Clear genre identification
   - Genre-appropriate conventions
   - Style characteristics
   - Cultural associations (if relevant)

2. **Mood and Atmosphere**
   - Emotional tone
   - Energy level
   - Atmosphere description
   - Emotional journey (if applicable)

3. **Tempo and Rhythm**
   - BPM specification
   - Time signature
   - Rhythm pattern description
   - Groove and feel

4. **Harmony and Melody**
   - Key specification
   - Scale or mode
   - Chord progression style
   - Melodic characteristics

5. **Instrumentation**
   - Primary instruments
   - Supporting instruments
   - Textural elements
   - Arrangement style

6. **Structure (if applicable)**
   - Song sections (verse, chorus, bridge)
   - Arrangement flow
   - Build-ups and releases
   - Dynamic changes

**Suno-Specific Formatting:**
- Use [GENRE] metatags when appropriate
- Include [BPM] specification
- Add [KEY] if relevant
- Use descriptive text for mood and atmosphere
- Structure with clear sections if needed

**Output Format:**
- Clear, structured prompt text
- Genre-appropriate metatags
- Specific technical specifications
- Descriptive mood and atmosphere
- 50-200 words typically

---

**Example Usage:**
```
Task: Create a Suno prompt for an upbeat electronic track
Genre: Electronic
Mood: Energetic, uplifting
BPM: 128
Key: C major
```
