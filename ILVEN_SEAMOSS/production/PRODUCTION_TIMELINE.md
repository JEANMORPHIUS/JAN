**Duration:** 4-6 hours

- Segment 3: The Work

## CONTINGENCY PLANNING

### Weather (Bus Shots)
- **Backup:** Indoor bus shots, or reschedule
- **Alternative:** Use existing bus footage if available

### Location Issues
- **Backup:** Alternative workshop location identified
- **Alternative:** Studio can substitute for some shots

### Equipment Failure
- **Backup:** Secondary camera/audio equipment
- **Alternative:** iPhone backup always available

### Talent Availability
- **Backup:** Voice-over can be recorded separately
- **Alternative:** Multiple recording sessions if needed

### Timeline Delays
- **Buffer:** Built-in buffer days in each phase
- **Flexibility:** Phases can overlap if needed
