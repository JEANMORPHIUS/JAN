# Production Day Execution Guide: Listen İlven

**Purpose:** Step-by-step execution guide for each production day  
**Philosophy:** Organized execution, authentic moments, heart first

---

## PRE-PRODUCTION DAY CHECKLIST

### 24 Hours Before Each Shoot Day
- [ ] Equipment tested and charged
- [ ] Storage cleared and formatted
- [ ] Location confirmed (time, address, contact)
- [ ] Weather checked (if outdoor elements)
- [ ] Team briefed (if applicable)
- [ ] Shot list reviewed
- [ ] Script reviewed
- [ ] Backup plans confirmed
- [ ] Equipment packed
- [ ] Transportation arranged

### Morning of Shoot Day
- [ ] Final equipment check
- [ ] Batteries at 100%
- [ ] Storage space verified
- [ ] Shot list printed/digital
- [ ] Script accessible
- [ ] Location details confirmed
- [ ] Arrival time planned (early is better)

## PRODUCTION DAY 1: WORKSHOP (Segments 1 & 3)

### Location: Workshop/Kitchen
### Duration: 4-6 hours
### Shots: Intro (The Graft) + The Work

---

### TIMELINE

#### 9:00 AM - Arrival & Setup (30 minutes)
- [ ] Arrive at location
- [ ] Meet contact, confirm access
- [ ] Unload equipment
- [ ] Scout space for best angles
- [ ] Set up base camp (equipment area)

#### 9:30 AM - Lighting & Camera Setup (30 minutes)
- [ ] Assess natural light
- [ ] Set up supplemental lighting (if needed)
- [ ] Position LED panels/reflectors
- [ ] Set up camera (iPhone)
- [ ] Test framing (vertical format)
- [ ] Test focus and exposure
- [ ] Set up microphone
- [ ] Test audio levels

#### 10:00 AM - Segment 1: Intro (The Graft) - 1 hour
**Shot 1.1: Narrator Close-Up (5 seconds)**
- [ ] Frame: Medium close-up, handheld
- [ ] Position narrator
- [ ] Test lighting on face
- [ ] Record 3-4 takes
- [ ] Review takes, select best
- [ ] Note: Natural lighting, warm tones

**Shot 1.2: Hands Working (5 seconds)**
- [ ] Frame: Macro shot, hands preparing sea moss
- [ ] Set up macro focus
- [ ] Position hands in frame
- [ ] Record 3-4 takes
- [ ] Review takes, select best
- [ ] Note: High texture, focus on craftsmanship

**Shot 1.3: Wide Workshop (5 seconds)**
- [ ] Frame: Wide angle, showing workspace
- [ ] Adjust camera position
- [ ] Frame entire workspace
- [ ] Record 3-4 takes
- [ ] Review takes, select best
- [ ] Note: Establish real-world setting

#### 11:00 AM - Break (15 minutes)
- [ ] Review footage
- [ ] Backup files (if time allows)
- [ ] Hydrate, rest

#### 11:15 AM - Segment 3: The Work - 1.5 hours
**Shot 3.1: Sea Moss Raw (4 seconds)**
- [ ] Frame: Macro, extreme close-up
- [ ] Set up product
- [ ] Focus on texture and color
- [ ] Record 3-4 takes
- [ ] Review takes, select best
- [ ] Note: Show natural beauty

**Shot 3.2: Preparation Process (6 seconds)**
- [ ] Frame: Medium, following hands
- [ ] Set up for process shots
- [ ] Record cleaning, cutting, mixing
- [ ] Multiple angles if needed
- [ ] Record 3-4 takes
- [ ] Review takes, select best
- [ ] Note: Documentary style, handheld

**Shot 3.3: Product Close-Up (5 seconds)**
- [ ] Frame: Macro, product in container
- [ ] Set up finished product
- [ ] Focus on texture and color
- [ ] Record 3-4 takes
- [ ] Review takes, select best
- [ ] Note: Hero shot of product

**Shot 3.4: Jar/Packaging (5 seconds)**
- [ ] Frame: Medium close-up
- [ ] Set up labeling/packaging
- [ ] Record process
- [ ] Record 3-4 takes
- [ ] Review takes, select best
- [ ] Note: Show care in presentation

#### 12:45 PM - B-Roll & Additional Coverage (30 minutes)
- [ ] Capture additional angles
- [ ] Detail shots (tools, textures)
- [ ] Establishing shots
- [ ] Cutaway shots
- [ ] Safety shots (extra coverage)

#### 1:15 PM - Audio Recording (30 minutes)
- [ ] Record ambient workshop sounds
- [ ] Water sounds
- [ ] Cutting sounds
- [ ] Stirring sounds
- [ ] Packaging sounds
- [ ] General ambience

#### 1:45 PM - Wrap & Backup (30 minutes)
- [ ] Review all footage
- [ ] Backup files to external drive
- [ ] Backup to cloud (if time allows)
- [ ] Pack equipment

## GENERAL PRODUCTION GUIDELINES

### During Shoots
- **Stay Organized:** Keep shot list accessible
- **Review Takes:** Check footage periodically
- **Backup Regularly:** Don't wait until end
- **Stay Hydrated:** Long days require energy
- **Be Flexible:** Adapt to conditions
- **Capture Authenticity:** Real moments over perfection

### Quality Control
- [ ] Check focus before each take
- [ ] Verify audio levels
- [ ] Review framing (vertical format)
- [ ] Check exposure
- [ ] Monitor storage space
- [ ] Backup critical shots immediately

### Communication
- [ ] Keep team informed (if applicable)
- [ ] Update location contacts
- [ ] Document any issues
- [ ] Note retakes needed

---

## POST-SHOOT DAILY TASKS

### End of Each Day
- [ ] Review all footage
- [ ] Backup to external drive
- [ ] Backup to cloud (if possible)
- [ ] Organize files by segment
- [ ] Update production notes
- [ ] Charge all equipment
- [ ] Prepare for next day

### Before Next Shoot
- [ ] Review previous day's footage
- [ ] Note any retakes needed
- [ ] Adjust plan if necessary
- [ ] Prepare equipment
- [ ] Confirm next location

---

## CONTINGENCY HANDLING

### Equipment Issues
- **Problem:** Camera/microphone failure
- **Solution:** Use backup equipment
- **Prevention:** Test everything beforehand

### Location Issues
- **Problem:** Location unavailable
- **Solution:** Use backup location
- **Prevention:** Confirm day before

### Weather Issues (Bus)
- **Problem:** Bad weather
- **Solution:** Reschedule or use indoor shots
- **Prevention:** Check forecast, have backup

### Time Issues
- **Problem:** Running behind schedule
- **Solution:** Prioritize essential shots
- **Prevention:** Build buffer time

---

**Remember:** Production days are about capturing authentic moments. Stay organized, but don't let process overshadow the craft. Film what is felt, not just what is seen. Heart first. Always heart first.
