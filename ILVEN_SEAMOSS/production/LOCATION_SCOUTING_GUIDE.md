# Location Scouting Guide: Listen İlven

**Purpose:** Secure authentic locations that serve the narrative  
**Philosophy:** Real spaces, real moments, real craft

## LOCATION 1: WORKSHOP/KITCHEN

### Requirements
- **Purpose:** Segments 1 & 3 (Intro & The Work)
- **Aesthetic:** Authentic, lived-in, craft-focused
- **Size:** Enough space for camera work and product preparation
- **Lighting:** Natural light preferred, window access ideal

### Key Features Needed
- [ ] Work surface for sea moss preparation
- [ ] Storage for tools and materials
- [ ] Sink/water access
- [ ] Natural lighting (windows)
- [ ] Texture-rich surfaces (wood, metal, organic materials)
- [ ] Space for wide shots and macro shots
- [ ] Minimal background noise
- [ ] Power outlets for equipment

### Scouting Checklist
- [ ] Location address and contact
- [ ] Access hours and availability
- [ ] Permission/rental requirements
- [ ] Parking/transportation
- [ ] Lighting conditions (time of day)
- [ ] Noise levels (traffic, neighbors)
- [ ] Safety considerations
- [ ] Backup location identified

### Ideal Locations
- Home kitchen with craft aesthetic
- Artisan workshop space
- Co-working kitchen
- Private studio with kitchen facilities
- Community kitchen space

### Questions to Ask
- Can we film during specific hours?
- Is there natural light available?
- Are there any restrictions on equipment?
- Is the space available for multiple days if needed?
- What's the background noise level?
- Are there any aesthetic restrictions?

### Backup Options
- Alternative workshop location
- Studio with kitchen setup
- Home-based setup

## LOCATION 2: 329 BUS ROUTE

### Requirements
- **Purpose:** Segment 2 (The Journey)
- **Route:** Wood Green → Enfield Lock
- **Aesthetic:** Real London bus, authentic commute
- **Access:** Public transport, no special permission needed

### Key Features Needed
- [ ] Route 329 bus access
- [ ] Window views of London streets
- [ ] Interior shots (narrator on bus)
- [ ] Route signage visible
- [ ] Natural lighting through windows
- [ ] Authentic bus sounds

### Scouting Checklist
- [ ] Verify 329 bus route still active
- [ ] Test journey time (Wood Green → Enfield Lock)
- [ ] Identify best time of day for lighting
- [ ] Note bus frequency
- [ ] Check for route signage visibility
- [ ] Test audio recording (bus sounds)
- [ ] Identify best seats for filming
- [ ] Plan for multiple journeys if needed

### Route Details
- **Start:** Wood Green Station/Bus Stop
- **End:** Enfield Lock Station/Bus Stop
- **Duration:** Approximately 20-30 minutes
- **Frequency:** Check TfL schedule
- **Best Times:** 
  - Morning (9-11 AM) for natural light
  - Avoid rush hour for space/comfort

### Filming Considerations
- Be respectful of other passengers
- Use discreet equipment
- Natural, authentic moments preferred
- May need multiple journeys for coverage
- Consider off-peak times for better access

### Backup Options
- Alternative bus route (if 329 unavailable)
- Indoor bus shots (if weather issues)
- Pre-recorded bus footage (if needed)

- [ ] Controlled lighting setup
- [ ] Space for camera and equipment

- [ ] Minimal background noise
- [ ] Power outlets

- [ ] Equipment included or BYO
- [ ] Lighting setup options
